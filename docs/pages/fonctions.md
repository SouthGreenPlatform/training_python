Les fonctions sont des __blocs de code isolés__, qui peuvent être __appelés__ depuis n'importe où dans votre code. 

Une fonction prend des __arguments__ en __entrée__, et peut renvoyer une __valeur de retour__ en sortie.

Syntaxe :

```python
x = fonction(argument1, argument2)
```

On peut appeler une fonction un nombre illimité de fois. Une fonction peut elle-même appeler d'autres fonctions.
