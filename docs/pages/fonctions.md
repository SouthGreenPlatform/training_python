# Les fonctions

Les fonctions sont des  _blocs de code isolés_ \, qui peuvent être  _appelés_  depuis n'importe où dans votre code\.

Une fonction prend des  _arguments_  en  _entrée_ \, et peut renvoyer une  _valeur de retour_  en  _sortie_ \.

Syntaxe : _x = fonction\(argument1\, argument2\)_

On peut appeler une fonction un nombre illimité de fois\. Une fonction peut elle\-même appeler d'autres fonctions\.
